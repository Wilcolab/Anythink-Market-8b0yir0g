- PascalCase input (e.g., 'HelloWorld' → 'hello-world')
- snake_case input (e.g., 'hello_world' → 'hello-world')
- Space-separated input (e.g., 'hello world' → 'hello-world')

   - Convert the entire result to lowercase
   - Clean up any duplicate hyphens
